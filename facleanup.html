FA Cleanup

A simple script that moves the new sidebar on FA back below the image
Update: Apparently FA will be removing the sidebar. When that does happen, I'll push out an update that should disable this plugin.

Installation

  1. If you're using Firefox, install Greasemonkey.
    If you're using Chrome or Safari, install Tampermonkey.
    If you're using Opera, install Violentmonkey.
  2. Visit http://boothale.net/scripts/facleanup.user.js and choose "Install" when prompted.
  3. Visit FA and ensure the script is working. If it isn't, try reinstalling it or restart your browser.

This script is ONLY intended for moving the sidebar. It does not remove or modify any advertisements on the site.
If you're trying to remove the large empty space in the sidebar, try the Element Hiding Helper.

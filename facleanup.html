FA Cleanup

Update: Apparently FA will be removing the sidebar. If you're running the latest version, the script should automatically disable when the sidebar goes away.

This is a simple script that moves the new sidebar on FA back below the image
It's ONLY intended for moving the sidebar. It does not remove or modify any advertisements on the site.
If you're trying to remove the large empty space in the sidebar, try the Element Hiding Helper.
Send any questions, comments or ramblings to erra@boothale.net.
Code is available on github.

Installation

  1. If you're using Firefox, install Greasemonkey.
    If you're using Chrome or Safari, install Tampermonkey.
    If you're using Opera, install Violentmonkey.
  2. Visit http://boothale.net/scripts/facleanup3.user.js and choose "Install" when prompted.
  3. Visit FA and ensure the script is working. If it isn't, try reinstalling it or restart your browser.

Removal

Firefox
You can remove Greasemonkey in the add-ons Manager.
To remove only this script, open the add-ons Manager, select "User Scripts" then click the "Remove" button next to the script.
Chrome or Safari
You can remove Tampermonkey by following the guide in their FAQ.
To remove only the script, click the Tampermonkey icon, select "Dashboard" and click the trashcan icon next to the script.
Opera
You can remove Violentmonkey using the Extensions Manager.
To remove only this script, click the Violentmonkey icon, select "Manage Scripts" then click the "Remove" button next to the script.
